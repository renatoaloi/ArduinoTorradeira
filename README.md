Arduino Torradeira

Arduino controlando uma resistência de torradeira a 70 graus celcius.

Este pacote de códigos contém o sketch do Arduino, os código CSharp e os esquemas de ligação.

Pastas e Arquivos

- Arduino: Código para carregar no Arduino
- Visual Studio 2010: Código CSharp para receber as informações via Serial da temperatura e apresentar em forma de gráfico
- schematics1.png: Ligações do Arduino no Dimmer, potenciometro e LM35
- schematics2.png: Ligações do Dimmer na resistência e na tomada

**en-US

Code and schematics for Arduino controlling a Toaster resistor at 70 celcius degrees.

This package contains Arduino's sketch, Csharp codes and wiring schematics.

Folders and Files

- Arduino: Arduino's sketch
- Visual Studio 2010: CSharp Winform application that receives serial temperature data from Serial and convert it to graph.
- schematics1.png: Wiring for Arduino, Dimmer, pot. and LM35
- schematics2.png: Wiring for Dimmer and Toaster resistor
